# lecture_PIV_postproc
Lecture on PIV postprocessing of a data from a planar jet.  Part of the course "Experimental Methods in Fluid Dynamics" taught at the University of Bologna.

Instructions for MATLAB:

Place the MATLAB scripts in the directory of the timeseries.

Follow the comments/instructions in the .m file.


Tutorial instructions for python:

In case of doubt, refer to the example dataset included with the notebook.

Use Binder to view the lesson as an executable - https://mybinder.org/ Alternatively, use a Jupyter Notebook reader to view the lecture. Eg. nbviewer https://nbviewer.org/
